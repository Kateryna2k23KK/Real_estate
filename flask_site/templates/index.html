Task: Predicting House Prices

The goal is to predict house prices based on various factors such as house size, number of rooms, property type, and other characteristics. The system should be able to process historical data on real estate prices and provide accurate predictions for new properties.

Goal

The goal of the project is to develop a system that predicts real estate prices based on multiple factors, including square footage, number of rooms, location, and other features. To achieve this, data needs to be collected and processed using machine learning methods, followed by training models (Linear Regression, Random Forest, and XGBoost) to accurately predict house prices based on these factors.

Additionally, the project includes creating an API that will accept input data from the user and return the predicted result. It is also important to evaluate the model's performance using accuracy metrics such as MAE and RMSE in order to choose the most suitable model for real-world conditions.
